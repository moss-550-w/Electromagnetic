# 电磁学科普静态网站设计方案
## 一、网站核心定位与设计理念
### 1. 核心定位
**"从琥珀摩擦到5G通信：一场跨越三千年的认知革命"**
- 以**科学史的曲折性**为灵魂，打破"科学发现是天才灵光一现"的神话
- 以**理论-工程的贯通**为骨架，展示"一个公式如何改变世界"
- 以**科学方法论的传递**为内核，让用户学会"像科学家一样思考"

### 2. 核心设计原则
- **真实优先**：不美化历史，不神化科学家，完整呈现争议、错误、偶然与压制
- **分层呈现**：普通用户看故事，爱好者看细节，专业用户看原始文献
- **沉浸式体验**：用交互让抽象的"场"和"波"变得可感知
- **启示导向**：每个知识点都对应一条可迁移的科学思维方法

### 3. 视觉设计系统
| 元素 | 设计规范 | 设计意图 |
|------|----------|----------|
| **主色调** | 深空蓝(#0F172A) + 电弧金(#F59E0B) + 磁石灰(#64748B) | 蓝色代表理性与宇宙，金色代表电的火花，灰色代表磁的沉稳 |
| **辅助色** | 正电红(#EF4444) + 负电蓝(#3B82F6) | 用于区分正负极、两种电荷，强化物理直觉 |
| **字体** | 标题：Inter Bold；正文：Inter Regular；公式：Latin Modern Math | 现代无衬线字体保证易读性，专业公式字体保证准确性 |
| **图标系统** | 手绘风格的历史实验装置图标 | 莱顿瓶、伏打电堆、法拉第圆盘、赫兹电火花等，增强历史感 |
| **页面风格** | 极简主义+学术感，大量留白，重点内容用金色高亮 | 避免视觉干扰，让用户专注于内容本身 |

## 二、网站整体架构
```
电磁学认知革命
├── 首页
├── 核心板块1：交互式电磁学发展时间线（灵魂板块）
├── 核心板块2：理论的诞生与争议（深度板块）
├── 核心板块3：从公式到世界：工程应用史（特色板块）
├── 核心板块4：凡人科学家列传（人文板块）
├── 核心板块5：科学启示录（价值板块）
└── 资源中心
```

## 三、各板块详细设计
### 1. 首页：沉浸式开场与核心入口
#### 1.1 开场动画（3秒，可跳过）
- 黑色背景中，一点金色火花闪烁，逐渐扩散成电场线
- 电场线交织成"琥珀→莱顿瓶→伏打电堆→发电机→手机"的演变路径
- 最终定格为网站标题：**"电磁学：一场跨越三千年的认知革命"**
- 副标题："每一个理所当然的公式背后，都是百年的争论与试错"

#### 1.2 首页核心区域
- **顶部导航栏**：固定在顶部，滚动时变透明，包含5个核心板块入口
- **中央核心入口**：一个巨大的交互式时间线缩略图，点击直接进入时间线板块
- **三大特色卡片**：
  1. 「被误解的天才」：展示欧姆、法拉第等被主流忽视的科学家
  2. 「改变世界的10个公式」：从库仑定律到麦克斯韦方程组
  3. 「如果没有电磁学」：脑洞式展示没有电的现代世界
- **科学启示滚动栏**：自动滚动展示从电磁学史中提炼的金句：
  - "科学不是真理的集合，而是不断逼近真理的过程"
  - "主流范式往往是最大的认知牢笼"
  - "错误的理论也能催生伟大的发现"
  - "10岁上大学的神童很多，但敢于钻进故纸堆的开尔文很少"

### 2. 交互式电磁学发展时间线（灵魂板块）
#### 2.1 整体设计
- 横向滚动时间线，分为5个时代，对应长河劫系列视频：
  - 「萌芽时代」（公元前600年-1600年）：从泰勒斯到吉尔伯特
  - 「静电时代」（1600年-1800年）：从格里克到库仑
  - 「电流时代」（1800年-1820年）：从伽伐尼到欧姆
  - 「统一时代」（1820年-1865年）：从奥斯特到麦克斯韦
  - 「应用时代」（1865年-至今）：从赫兹到5G
- 每个时代用不同颜色的时间轴区分，时间节点用圆形图标表示，重要节点用金色大图标突出

#### 2.2 时间节点交互设计
点击任意时间节点，弹出分层信息卡片：
- **第一层（默认显示）**：
  - 时间、人物、核心事件一句话总结
  - 一张历史图片或实验装置图
  - **「历史真相」标签**：用红色字体标注最颠覆认知的事实
    - 例：奥斯特的发现是讲座结束后的偶然尝试，并非精心设计的实验
    - 例：欧姆定律发表后被骂了10年，被称为"纯粹的数学虚构"
- **第二层（点击"展开详情"）**：
  - 完整的事件经过
  - 当时的主流观点与反对意见
  - 这个发现的直接影响
- **第三层（点击"科学启示"）**：
  - 从这个事件中提炼的科学思维方法
  - 例：伽伐尼的"生物电"错误→"错误假说的价值在于引导实验"
  - 例：开尔文研究法拉第的笔记→"独立思考比追随主流更重要"
- **关联跳转**：每个节点都有链接跳转到对应的理论专题、工程案例和科学家列传

#### 2.3 创意功能：「平行宇宙时间线」
- 点击时间线上的关键分歧点，进入"平行宇宙"：
  - 如果富兰克林的风筝实验被雷劈中了怎么办？
  - 如果奥斯特没有做那次偶然的实验，电磁学会推迟多少年？
  - 如果麦克斯韦没有提出位移电流假说，电磁理论会如何发展？
- 每个平行宇宙都基于真实的历史逻辑推演，展示科学发现的偶然性

### 3. 理论的诞生与争议（深度板块）
#### 3.1 板块结构
按理论重要性排序，每个理论一个独立页面，结构统一为：
```
[理论名称]
├── 诞生前夜：当时的困境与主流观点
├── 突破时刻：发现的完整过程
├── 核心内容：公式+图解+通俗解释
├── 世纪之争：当时的反对意见与争论焦点
├── 最终胜利：理论被接受的过程
├── 工程遗产：这个理论催生了哪些技术
└── 科学启示
```

#### 3.2 创意子板块：「错误理论博物馆」
这是整个网站最有特色的部分，专门展示那些被推翻的理论：
- **动物电假说**：伽伐尼的青蛙实验与伏打的反驳
- **单流体说vs双流体说**：富兰克林与杜菲的百年争论
- **超距作用理论**：牛顿力学的遗产与法拉第的挑战
- **以太假说**：麦克斯韦时代的"绝对参考系"与迈克尔逊-莫雷实验
- 每个错误理论都包含：
  - 理论的核心内容
  - 为什么当时的人会相信它
  - 它是如何被推翻的
  - 它对科学发展的积极贡献
- 板块引言："科学史就是一部错误理论的墓志铭，但每一块墓碑上都刻着进步的脚印"

#### 3.3 创意子板块：「公式的诞生」
- 不是简单罗列公式，而是拆解每个公式的"前世今生"
- 以麦克斯韦方程组为例：
  - 第一个方程（高斯电场定律）：来自库仑的实验
  - 第二个方程（高斯磁场定律）：来自磁单极不存在的观察
  - 第三个方程（法拉第电磁感应定律）：来自法拉第的10年实验
  - 第四个方程（安培环路定理+位移电流）：安培的实验+麦克斯韦的天才假设
- 用动画展示麦克斯韦如何"凭空"添加位移电流项，以及这一项如何预言了电磁波的存在

### 4. 从公式到世界：工程应用史（特色板块）
#### 4.1 板块设计理念
打破"科学归科学，工程归工程"的割裂，展示**"一个理论→一个发明→一个产业→改变人类生活"**的完整链条

#### 4.2 核心案例设计（每个案例一个独立页面）
以「发电机与电网」为例：
1. **理论源头**：1831年法拉第电磁感应定律
   - 展示法拉第的圆盘发电机原始图纸
   - 说明为什么圆盘发电机没有实用价值
2. **技术突破**：1866年西门子发明自激式发电机
   - 对比西门子发电机与法拉第圆盘的区别
   - 讲解自激式的原理：用发电机自己发的电给电磁铁供电
3. **产业革命**：从直流电到交流电的战争
   - 爱迪生的直流电系统与特斯拉的交流电系统
   - 为什么交流电最终胜出（变压器的发明）
4. **现代应用**：三峡水电站与全球电网
   - 展示现代发电机的结构图
   - 讲解电网的工作原理
5. **未来展望**：核聚变发电与无线输电
6. **科学启示**："基础研究的价值，往往在几十年后才会显现"

#### 4.3 创意功能：「如果没有电磁学」互动模拟器
- 用户可以选择"禁用"一个电磁学发现，看看世界会变成什么样：
  - 禁用电磁感应：没有电，没有电灯、电视、手机，回到蒸汽时代
  - 禁用电磁波：没有无线电、电视、手机、互联网
  - 禁用麦克斯韦方程组：没有现代电子工程，所有电子设备都不存在
- 用图文并茂的方式展示后果，让用户直观感受到基础科学的力量

### 5. 凡人科学家列传（人文板块）
#### 5.1 板块设计理念
不做"天才神话"，而是展示**"有缺点、有困惑、会犯错、会被误解的凡人科学家"**，让用户觉得"他们和我一样，只是更坚持、更独立思考"

#### 5.2 科学家页面结构
```
[科学家姓名]
├── 凡人画像：他们的性格、缺点与人生困境
├── 研究之路：不是一帆风顺，而是充满挫折
├── 核心贡献：客观评价，不夸大
├── 他们的争议：当时的人如何评价他们
├── 留给后世的启示
└── 他们的名言
```

#### 5.3 重点人物设计
- **迈克尔·法拉第**：
  - 重点讲他的出身（铁匠的儿子，学徒）
  - 他的数学短板（只会加减乘除，不会微积分）
  - 他的"场"概念如何被主流嘲笑为"玄学"
  - 启示："直觉和想象力，有时比数学更重要"
- **开尔文（威廉·汤姆逊）**：
  - 重点讲他不盲从主流的精神
  - 他如何钻进法拉第的"故纸堆"，把"力线"转化为数学
  - 他的晚年失误（两朵乌云）
  - 启示："科学研究不是附和主流，而是独立判断"
- **詹姆斯·麦克斯韦**：
  - 重点讲他的想象力（用流体力学类比电磁场）
  - 他的理论在生前没有被广泛接受
  - 他的早逝（48岁去世，没有看到赫兹的实验）
  - 启示："伟大的理论往往超越时代"
- **海因里希·赫兹**：
  - 重点讲他的实验如何验证了电磁波
  - 他的著名误判："电磁波没有任何实际用途"
  - 启示："即使是最伟大的科学家，也无法预见未来"

#### 5.4 创意子板块：「被遗忘的先驱」
专门介绍那些贡献被低估的科学家：
- 斯蒂芬·格雷：发现导体与绝缘体，被皇家学会忽视
- 乔治·欧姆：欧姆定律的提出者，被主流压制10年
- 楞次：楞次定律的提出者，很少被人记住
- 毕奥与萨伐尔：毕奥-萨伐尔定律的提出者
- 板块引言："科学史是由胜利者书写的，但每一个胜利者的背后，都有无数被遗忘的铺路石"

### 6. 科学启示录（价值板块）
这是整个网站的升华部分，从电磁学史中提炼出普适的科学方法论和人生哲学。

#### 6.1 核心内容
- **科学发现的5条规律**：
  1. 科学不是线性进步，而是螺旋式上升
  2. 错误是科学进步的必经之路
  3. 偶然发现只青睐有准备的头脑
  4. 主流范式往往是最大的认知障碍
  5. 实验是检验真理的唯一标准
- **给年轻人的5条建议**：
  1. 不要盲从权威，要独立思考
  2. 不要害怕犯错，要从错误中学习
  3. 不要忽视"无用"的基础研究
  4. 要像开尔文一样，去故纸堆里找宝藏
  5. 保持好奇心，它是科学研究的原动力
- **科学与社会**：
  - 科学研究需要宽容的环境
  - 基础研究的价值不能用短期利益衡量
  - 科学是全人类的共同财富

#### 6.2 创意设计：「电磁学史的10个教训」
用卡片式设计，每个教训配一张历史图片和一句精炼的总结：
1. 泰勒斯：好奇是科学的起点
2. 吉尔伯特：实验是科学的基础
3. 格雷：小人物也能做出大发现
4. 伽伐尼：错误的理论也有价值
5. 伏打：质疑权威是科学的灵魂
6. 欧姆：真理需要时间来证明
7. 奥斯特：偶然中蕴含着必然
8. 法拉第：想象力比知识更重要
9. 开尔文：独立思考胜过追随主流
10. 麦克斯韦：理论思维能预见未来

### 7. 资源中心
- **视频推荐**：长河劫电磁学系列、BBC《科学的历史》、费曼访谈等
- **书籍推荐**：《电磁学发展史》《法拉第传》《麦克斯韦传》等
- **原始文献库**：链接到法拉第的实验笔记、麦克斯韦的原始论文等
- **实验复刻指南**：10个在家就能做的简单电磁学实验（附材料清单和步骤）
- **高清素材下载**：电磁学发展时间线海报、麦克斯韦方程组海报、科学家肖像等

## 四、创意交互设计（纯前端实现，无需后端）
1. **电场线可视化**：用户可以在页面上拖动正电荷和负电荷，实时看到电场线的变化
2. **电磁波传播动画**：用SVG动画展示电磁波的传播过程，用户可以调整频率和振幅
3. **电磁学认知闯关**：10道选择题，每道题对应一个历史事件，逐题作答即时解析，答对 **6 题及以上**获得"电磁学历史达人"证书（可下载）。〔实现决策：发证门槛设为 6/10——原"全部答对"过严、易挫败，放宽以提高完成率与科普覆盖面；门槛常量 `PASS_THRESHOLD` 集中在 `src/data/quiz.ts`，可随时调整〕
4. **时间线对比**：同时展示电磁学发展和同时代的世界历史事件，让用户有更宏观的视角
5. **公式拆解动画**：点击麦克斯韦方程组的任意一项，都会弹出动画解释其物理意义

## 五、内容分层设计
为了满足不同用户的需求，所有内容都分为三个层次：
- **快速阅读层**：用金色字体标注核心结论，适合普通用户快速浏览
- **深度阅读层**：完整的历史过程、理论解释和工程应用，适合爱好者
- **专业研究层**：原始文献链接、公式推导过程、实验数据，适合专业用户

## 六、SEO与可扩展性设计
- 每个页面都有独立的标题、描述和关键词，针对搜索引擎优化
- URL结构清晰，采用`/板块/文章名`的形式
- 所有图片都添加alt标签
- 网站采用模块化设计，未来可以轻松添加新的内容板块（如量子电磁学、超导技术等）
- 支持多语言扩展，未来可以添加英文版本

## 七、科学严谨性保障
1. 所有历史事件都有可靠的史料来源，标注出处
2. 所有物理公式和理论都经过严格验证
3. 对于有争议的历史问题，同时展示不同观点
4. 避免过度演绎和虚构，所有"脑洞"都基于真实的历史逻辑
